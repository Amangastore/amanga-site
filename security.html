Security & Compliance

We align with Amazon’s Acceptable Use and Data Protection policies.

Access control

  • Least‑privilege access; role‑based permissions; access reviews every 90 days.
  • Secrets are stored in a vault (environment secrets), no public sharing.

Encryption & transport

  • TLS for all network communications.
  • Encrypted storage for tokens and backups.

Incident response

  • We monitor and log failures.
  • Security incidents involving Amazon Information will be reported to security@amazon.com without undue delay; target response ≤ 72 hours.

Data handling

  • No PII from Amazon Buyers is collected.
  • Amazon Information is used solely for our own seller account operations.
  • Data retention is minimal and deleted when no longer needed (max 90 days unless legally required).
